# Algo Trading Platform 🚀

An AI-driven algorithmic trading web app using LSTM and FinBERT. Built with FastAPI, Streamlit, and PostgreSQL.

## 🔧 Tech Stack
- Frontend: Streamlit
- Backend: FastAPI (Python)
- ML: LSTM, FinBERT
- Database: PostgreSQL

## 🧩 Features
- Trading Dashboard
- Market Sentiment and Price Predictions
- Order Execution and Management

## 📁 Folder Structure
- `frontend/`: UI components (Streamlit)
- `backend/`: FastAPI + ML models
- `database/`: SQL and connection scripts
- `notebooks/`: ML model training
