# Algo Trading Platform 🚀

An AI-driven algorithmic trading web app using LSTM for stock price prediction. Built with FastAPI, Streamlit, and PostgreSQL.

## 🔧 Tech Stack
- Frontend: Streamlit
- Backend: FastAPI (Python)
- ML: LSTM
- Database: PostgreSQL

## 🧩 Features
- stock Price Predictions


## 📁 Folder Structure
- `frontend/`: UI components (Streamlit)
- `backend/`: FastAPI + ML models
- `database/`: SQL and connection scripts
- `notebooks/`: ML model training
